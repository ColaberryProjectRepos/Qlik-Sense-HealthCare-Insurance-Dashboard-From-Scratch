Why use $():

So when we used SET, it just returns the string and does not evaluate the command.  This is why we use the $(). 

for example we 

Set vAge= 22*3

when we use a chart and use that variable as an expression then we will get 22*3

If we have the same scenario and use the expression as $(22*3), we will get 66.
