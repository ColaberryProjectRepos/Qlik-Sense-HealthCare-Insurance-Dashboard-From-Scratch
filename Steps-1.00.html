How to Acquire the Source Data

 

The 1st step is to Locate the Data by Clicking on the Data File (on this page).

 

The next task is to take the downloaded file (QVD Files), copy and paste it to the Local C Drive file.

--Locate the C drive; depending on how your computer is set up it may look similar to this:

 

--Inside the C drive, just right click on the empty space and select Paste


 





The source for the Dashboard is coming from a SQL Server database known as HealthCare Insurance.  It houses 3 Dimension ( Dim_Member_Details, Dim_Group_Details, Dim_Plan_Details) and 1 Fact Table (Fact_HealthCare).
